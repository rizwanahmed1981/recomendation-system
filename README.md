# Recommendation System

A simple and practical **Recommendation System** built using Python.  
This project is focused on understanding the core ideas behind recommendation engines, like how data is processed and how recommendations are generated.

Well, recommendation systems are everywhere, right?  
Netflix, YouTube, Amazon. So building one is a great learning move.
